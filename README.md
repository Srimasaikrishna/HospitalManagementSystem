# Hospital Management System (HMS) - Spring Boot

An **end-to-end Hospital Management System** developed using **Java Spring Boot**. This application provides a complete solution for managing hospital operations efficiently, including patient management, doctor management, appointments, billing, pharmacy, and more.

## Features

- **Patient Management**: Add, update, view, and manage patient records.
- **Doctor Management**: Add, update, and view doctor profiles and schedules.
- **Appointment Management**: Schedule and manage patient appointments with doctors.
- **Billing System**: Generate invoices for treatments and services.
- **Pharmacy Management**: Manage medicines and pharmacy inventory.
- **Reports**: Generate basic reports for hospital operations.

## Technology Stack

- **Backend**: Java, Spring Boot, Spring MVC, Spring Data JPA, Hibernate
- **Database**: MySQL
- **Frontend**:  React
- **Build Tool**: Maven
- **Security**: Spring Security
